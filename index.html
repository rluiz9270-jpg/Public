O HTML é a linguagem de marcação. Ele não é uma linguagem de programação, mas sim a estrutura básica de qualquer página na web.
O CSS é a linguagem de estilos. Ele trabalha diretamente sobre o HTML para transformá-lo visualmente.

18. O que é responsividade e por que ela é importante no desenvolvimento web?
A responsividade é a capacidade de um site adaptar seu layout e conteúdo automaticamente ao tamanho da tela em que é exibido (celular, tablet ou desktop). Em vez de criar sites diferentes para cada aparelho, utiliza-se um único código que "se molda" ao espaço disponível. Sua importância é crítica porque a maioria dos acessos atuais ocorre via smartphones; um site responsivo garante uma boa experiência ao usuário (evitando o "zoom" manual), melhora o ranqueamento no Google (que prioriza sites amigáveis ao mobile) e reduz custos de manutenção ao centralizar o projeto em uma única versão funcional para todos os dispositivos. 19. Explique o que é uma aplicação web moderna.
Uma aplicação web moderna (ou Web App) é um software que roda diretamente no navegador, mas que oferece uma experiência tão fluida, rápida e rica quanto a de um aplicativo instalado no seu celular ou computador.

Diferente dos sites antigos, onde cada clique exigia que a página inteira recarregasse (ficando branca por alguns segundos), as aplicações modernas são dinâmicas.

20. Cite duas boas práticas no desenvolvimento web e explique sua importância.
O HTML semântico consiste em usar as tags corretas para cada conteúdo; o que é vital para a acessibilidade (leitores de tela para deficientes visuais) e para o SEO, ajudando o Google a entender a estrutura do site.
Já a otimização de performance — que inclui a compactação de imagens e minificação de arquivos — é crucial para garantir que a página carregue rapidamente; sites lentos frustram o usuário e apresentam altas taxas de abandono, prejudicando a conversão e a retenção de visitantes.
